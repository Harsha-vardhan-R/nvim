To Use:
  * Install [Packer](https://github.com/wbthomason/packer.nvim) for nvim
  * Copy this repo into ~/.config/nvim
  * Open ~/.config/nvim/lua/hava/packer.lua
  * source it using `:so`
  * Install the plugins using `:PackerSync`
